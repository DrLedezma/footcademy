Política de Privacidad

FootCademy® Clinical Suites

Última actualización: 23 de marzo de 2026
Resumen: FootCademy® almacena los datos clínicos únicamente en el dispositivo del usuario (localStorage). No recopilamos, transmitimos ni vendemos información de pacientes a terceros. El único servicio en la nube utilizado es Firebase Authentication para gestión de acceso de usuarios.

1. Información General

FootCademy® Clinical Suites (en adelante, "la Aplicación") es una herramienta digital de apoyo académico y clínico desarrollada por SOMEPOMED — Sociedad Mexicana de Podología Médica A.C.

Esta Política de Privacidad describe cómo recopilamos, utilizamos y protegemos la información de los usuarios de la Aplicación.

3. Datos que Recopilamos

3.1 Datos de autenticación (Firebase)

Para acceder a la Aplicación, los usuarios deben crear una cuenta. Recopilamos:

  • Correo electrónico
  • Nombre del usuario
  • Rol asignado (administrador, ayudante, alumno)
  • Fecha de registro y última conexión

Estos datos se almacenan en Firebase (Google Cloud) con cifrado en tránsito y en reposo.

3.2 Datos clínicos de pacientes

La Aplicación permite capturar datos clínicos de pacientes como parte de evaluaciones podológicas. Estos datos incluyen, pero no se limitan a:

  • Nombre, edad, sexo, peso, talla
  • Valores de laboratorio (glucosa, HbA1c, perfil lipídico, etc.)
  • Clasificaciones clínicas (Wagner, Texas, WIfI, CEAP, etc.)
  • Diagnósticos CIE-10
  • Planes de manejo
IMPORTANTE: Todos los datos clínicos de pacientes se almacenan exclusivamente en el dispositivo del usuario mediante localStorage del navegador. No se transmiten a ningún servidor externo. No tenemos acceso a estos datos.

3.3 Datos que NO recopilamos

  • No recopilamos datos de ubicación (GPS)
  • No recopilamos datos de contactos del dispositivo
  • No recopilamos fotografías ni archivos del dispositivo
  • No utilizamos cookies de terceros ni rastreadores publicitarios
  • No recopilamos datos biométricos

4. Cómo Utilizamos los Datos

Tipo de dato Uso Almacenamiento
Correo y nombre Autenticación y gestión de acceso Firebase (Google Cloud)
Rol de usuario Control de permisos en la app Firebase Firestore
Datos clínicos Evaluación podológica del paciente localStorage (solo dispositivo)
PDF generados Reportes clínicos descargables Solo dispositivo del usuario

5. Compartición de Datos

No vendemos, alquilamos ni compartimos datos personales ni clínicos con terceros.

Los únicos servicios de terceros utilizados son:

  • Firebase Authentication y Firestore (Google): para gestión de cuentas de usuario. Sujeto a la Política de Privacidad de Firebase.
  • GitHub Pages (Microsoft): para alojar la aplicación web. No se transmiten datos de pacientes a este servicio.

6. Seguridad de los Datos

Implementamos las siguientes medidas de seguridad:

  • Autenticación obligatoria con correo y contraseña
  • Roles de usuario con permisos diferenciados
  • Comunicaciones cifradas mediante HTTPS/TLS
  • Datos clínicos almacenados solo localmente (no en la nube)
  • Suscripciones con fecha de caducidad configurable
  • Firebase Security Rules para proteger datos de usuarios

7. Retención de Datos

  • Datos de autenticación: Se conservan mientras la cuenta esté activa. El administrador puede desactivar o eliminar cuentas.
  • Datos clínicos: Permanecen en el dispositivo del usuario hasta que éste los elimine manualmente (limpiando localStorage o desinstalando la app).

8. Derechos del Usuario

De acuerdo con la Ley Federal de Protección de Datos Personales en Posesión de los Particulares (LFPDPPP) de México, usted tiene derecho a:

  • Acceso: Conocer qué datos personales tenemos sobre usted
  • Rectificación: Solicitar la corrección de sus datos
  • Cancelación: Solicitar la eliminación de sus datos
  • Oposición: Oponerse al tratamiento de sus datos

Para ejercer estos derechos (derechos ARCO), envíe un correo a: somepomed@gmail.com

9. Menores de Edad

La Aplicación está dirigida a profesionales de la salud y estudiantes de podología médica. No está diseñada para ser utilizada por menores de 18 años sin supervisión de un profesional.

10. Uso Académico y Clínico

FootCademy® Clinical Suites es una herramienta de apoyo académico y clínico. No sustituye el juicio clínico profesional, la exploración física ni los protocolos institucionales. Las dosis y recomendaciones son orientativas: validar contraindicaciones, interacciones y función renal/hepática del paciente.
